# YueduBookSource
我的阅读app有4000+书源，导出、去重（域名唯一）后的结果。

此页面github地址为： https://github.com/yoval/YueduBookSource ，

gitee地址为： https://gitee.com/fuwenyue/YueduBookSource 。

书源失效或者有新书源请提交ISSUES。

添加书源方法：我的→书源管理→网络导入。

1、导入此页全部书源：

(github) https://yoval.github.io/YueduBookSource/BookSource.json ,

(gitee)  http://fuwenyue.gitee.io/yuedubooksource/BookSource.json 。

二维码导入全部书源：

![gitee QRCODE](http://imgs.bizha.top/BookSource.png)

2、导入单个书源： 

(github) https://yoval.github.io/YueduBookSource/SingleBookSource/网站域名.json ，

(gitee)  http://fuwenyue.gitee.io/yuedubooksource/SingleBookSource/网站域名.json 。

如： 

(Github) https://yoval.github.io/YueduBookSource/SingleBookSource/kunnu.com.json

(Gitee)  http://fuwenyue.gitee.io/yuedubooksource/SingleBookSource/kunnu.com.json
