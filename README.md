# YueduBookSource
我的阅读app有4000+书源，导出、去重（域名唯一）后的结果。

添加书源方法：我的→书源管理→网络导入。

1、导入此页全部书源：https://yoval.github.io/YueduBookSource/BookSource.json 。

2、导入单个书源： 
https://yoval.github.io/YueduBookSource/网站域名/BookSource.json ，
如： https://yoval.github.io/YueduBookSource/kunnu.com/BookSource.json

书源失效或者有新书源请提交ISSUES。
