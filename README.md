# YueduBookSource
我的阅读app有4000+书源，导出、去重（域名唯一）后的结果。此页面github地址为： https://github.com/yoval/YueduBookSource ，gitee地址为： https://gitee.com/fuwenyue/YueduBookSource 。

添加书源方法：我的→书源管理→网络导入。

1、导入此页全部书源：
(github) https://yoval.github.io/YueduBookSource/BookSource.json ,
(gitee)  http://fuwenyue.gitee.io/yuedubooksource/BookSource.json 。


2、导入单个书源： 
(github) https://yoval.github.io/YueduBookSource/网站域名/BookSource.json ，
(gitee)  http://fuwenyue.gitee.io/yuedubooksource/网站域名/BookSource.json 。

如： 
(Github) https://yoval.github.io/YueduBookSource/kunnu.com/BookSource.json
(Gitee)  http://fuwenyue.gitee.io/yuedubooksource/kunnu.com/BookSource.json



二维码导入全部书源：
![gitee QRCODE](http://imgs.bizha.top/BookSource.png)






书源失效或者有新书源请提交ISSUES。
